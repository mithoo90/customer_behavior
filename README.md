Excited to share one of my recent portfolio projects where I built a complete **data analytics pipeline** using **Python, PostgreSQL, and Power BI**.

🔹 **Tools & Technologies Used**
Python (Pandas) – Data Cleaning & ETL
PostgreSQL – Exploratory Data Analysis (SQL)
Power BI – Data Visualization & Dashboarding

🔹 **Project Workflow**
Raw Dataset → Python ETL → PostgreSQL Analysis → Power BI Dashboard

🔹 **Key Findings from the Analysis**

• **73% of customers are non-subscribers**, indicating a large opportunity for subscription-based marketing strategies.
• **Male customers generated higher total spending compared to female customers.**
• **Sales drop during the winter season**, suggesting possible seasonal demand shifts or product mismatch.
